About

vin5m0k3sama is the first-party content distribution tool behind the Wait, Actually channel — a short-form video series that covers surprising facts about food, brands, and everyday products in under 60 seconds.

The tool handles the full publishing pipeline: it takes a finished MP4 video and distributes it to the channel's own social accounts using each platform's official, approved API. No third-party services, no unofficial automation — every upload goes through the platform's own developer API with OAuth authorization.

vin5m0k3sama only ever acts on behalf of the account owner, posting the owner's own original content to the owner's own accounts. It does not collect data from other users, does not post on anyone else's behalf, and does not bypass any platform controls.

How it works

1

Video ready

An original short-form MP4 video (vertical 9:16, under 60 seconds) is prepared by the content creator.

2

Caption generation

Platform-specific titles, descriptions, and hashtags are generated for YouTube, TikTok, Instagram, and Facebook.

3

API upload

The tool authenticates with each platform's official API using stored OAuth tokens and uploads the video with its metadata.

4

Published

The video goes live on all connected platforms. Upload results and any errors are logged for the account owner to review.

Supported platforms

YouTube

Uploads via YouTube Data API v3. Supports Shorts format, custom titles, descriptions, tags, and privacy settings.

TikTok

Publishes via TikTok Content Posting API (Direct Post). Uses scope video.publish only.

Instagram

Posts Reels via Instagram Graph API with Instagram Login. Supports captions and hashtags optimized for Reels reach.

Facebook

Publishes to the Wait, Actually Facebook Page via Graph API. Videos under 60s are classified as Reels automatically.

FAQ

What data does vin5m0k3sama access?

Only the minimum required to upload a video to the connected account: an OAuth access token and the account's allowed privacy settings. No personal data from other users is ever accessed or stored.

Does it post on behalf of other users?

No. vin5m0k3sama is a strictly first-party tool. It only posts content created by and belonging to the account owner, to the account owner's own accounts.

How are access tokens stored?

OAuth tokens are stored locally on the operator's machine in local files. They are never transmitted to any server other than the platform that issued them (e.g. TikTok or YouTube).

Can I revoke access?

Yes. You can revoke the app's access at any time from each platform's security settings. For TikTok: Settings → Security → Manage app permissions. For Google: myaccount.google.com → Security → Third-party apps.

Is this available to other users?

No. vin5m0k3sama is a private, single-operator tool. It is not a public service and does not offer accounts or sign-ups to third parties.
